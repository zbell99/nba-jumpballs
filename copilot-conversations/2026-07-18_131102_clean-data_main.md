## Summary of Implementation

### Functions Created

1. **`collect_dupes(df)`**
   - Identifies all groups of consecutive game plays (2+ sequential plays)
   - Returns list of dictionaries with: `game_id`, `start_game_play_number`, `end_game_play_number`, `length`
   - More efficient than original approach limited to 2-4 consecutive plays

2. **`clean_dupes_by_sparsity(df, consecutive_groups)`**
   - Removes sparse rows from consecutive groups using strict subset logic
   - Combines athlete_id_1, athlete_id_2, and athlete_id_3 into sets (nulls dropped)
   - Only removes a row if its athlete ID set is a strict subset of another row's set
   - Preserves rows with different (non-subset) athlete IDs

### Key Logic
- Uses Python's `<` operator for strict subset comparison
- Iterates through consecutive groups and performs pairwise athlete set comparisons
- If any row's set is a strict subset of another row's set, that row is marked for removal
